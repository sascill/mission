# Product Table

상품 테이블 기능과 관련된 타입, 목데이터, 컴포넌트를 관리하는 폴더입니다.

## 목적

- 상품 정보를 테이블 형태로 표시하기 위한 기능 단위를 분리한다
- 타입, 데이터, UI를 기능 기준으로 함께 관리한다

## 현재 구성

- `types/product.ts`: 상품 타입 정의
- `data/products.ts`: 상품 목데이터 10개

## 데이터 필드

- 상품ID
- 상품명
- 카테고리
- 가격
- 재고수량
- 상태
- 등록일

## 현재 완료 범위

- `Product` 타입 정의
- 상품 카테고리, 상태 타입 정의
- 10개의 상품 목데이터 export

## 제외 범위

- 테이블 UI 구현
- 페이지 레이아웃 구성
- 정렬, 검색, 필터, 페이징 기능

## 다음 작업

- 대시보드 페이지에 테이블 배치
